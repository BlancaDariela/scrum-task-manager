# Equipo 5 - Estadísticas

##  Objetivo

Generar métricas de las tareas.

##  Tareas

* Contar total de tareas
* Contar tareas completadas
* Calcular porcentaje
* Promedio de longitud
* Resumen general

##  Endpoints

* GET /team5/stats/total
* GET /team5/stats/completed
* GET /team5/stats/percentage
* GET /team5/stats/avg-length
* GET /team5/stats/summary

##  Reglas

* No modificar datos
* Solo cálculos
