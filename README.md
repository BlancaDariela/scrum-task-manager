# Equipo 1 - CRUD de Tareas

## 🎯 Objetivo

Implementar las operaciones básicas de las tareas.

## 🧠 Tareas

* Crear tarea
* Obtener todas las tareas
* Obtener tarea por ID
* Actualizar tarea
* Eliminar tarea

## 🔌 Endpoints

* POST /team1/tasks
* GET /team1/tasks
* GET /team1/tasks/{id}
* PUT /team1/tasks/{id}
* DELETE /team1/tasks/{id}
