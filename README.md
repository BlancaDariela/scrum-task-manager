# Equipo 2 - Listado y Ordenamiento

## Objetivo

Mostrar tareas de diferentes formas.

##  Tareas

* Listar tareas
* Ordenar por nombre
* Ordenar por longitud del nombre
* Limitar resultados
* Listado avanzado

##  Endpoints

* GET /team2/tasks/list
* GET /team2/tasks/sorted/name
* GET /team2/tasks/sorted/length
* GET /team2/tasks/limit
* GET /team2/tasks/advanced

##  Reglas

* No modificar la base de datos
* Solo consultas
* Usar parámetros en endpoints
