# Equipo 4 - Búsqueda y Filtros

##  Objetivo

Permitir búsquedas de tareas.

##  Tareas

* Buscar por nombre
* Buscar por palabra clave
* Filtrar por estado
* Búsqueda combinada
* Búsqueda avanzada

##  Endpoints

* GET /team4/tasks/search
* GET /team4/tasks/search/keyword
* GET /team4/tasks/filter
* GET /team4/tasks/search/advanced

##  Reglas

* Usar query params
* Manejar búsquedas vacías
